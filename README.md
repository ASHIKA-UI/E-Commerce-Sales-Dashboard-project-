# E-Commerce Sales Dashboard

## Problem
An online store wants to understand which products, regions, 
and time periods are driving revenue.

## Tools Used
Python | SQL (MySQL) | Excel | Power BI

## Key Findings
- Technology is the top category with 1M+ in sales
- West region leads with 29.55% of total sales
- December consistently peaks in monthly sales
- Medium discounts (20-40%) result in negative profit margins
- Canon imageCLASS 2200 is the top product by profit
